Issue&Solution;

Q1 : 在修改網站的時候有許多內容被吃掉或是覆蓋掉 ?

A1 : 項目名稱及Heading不可以重覆到，不然會造成覆蓋會者遺失的問題。

-------------------------------------------------------------------------------------------------------------

Q2 : git pull 完後導致全部跑版，先前已經改完的也受到影響 ?

A2 : 要等全部pull完成後才能add，否則會合併到新的，導致舊版已完成部分也受到影響。

-------------------------------------------------------------------------------------------------------------Q3 : 在網站上修改標題Experience內的子目錄時，標題Week11 Exp. 靜態動態近端都沒有出現錯誤，setting也沒有出現錯誤，但相對地改成Week10 Exp. 卻在網站上出現404 ?

A3 : 已將最新版本更新，衝突已解決。

-------------------------------------------------------------------------------------------------------------

Q4 : report資料夾改完檔名後在 github 上變成子目錄，不知道為什麼發生衝突了 ?

A4 : 因為有東西 ignore report資料夾 要避開這問題只要將 report 改成 report2。

------------------------------------------------------------------------------------------------------------

Q5 : 更新完 PDF 卻推不上去 ?

A5 : 因為PDF在 report 內，因為是子目錄所以會導致讀不到新增資料，因此無法addck和push。

------------------------------------------------------------------------------------------------------------

Q6 : 更新完blog之後，卻沒有更新跑出404 ?

A6 : 在 finalproject > blog 下沒有 index.html 的作業程序導致無法順利轉成網頁，導致這原因發生的事因為 md檔 衝突到導致舞法執行。
